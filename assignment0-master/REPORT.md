## Deskripsi
assigment 0 bertujuan membuat program sederhana menggunakan bahasa C++. Program yang dibuat merupakan modifikasi dari program "Hello World" agar dapat menerima input dari pengguna dan menampilkan nama pengguna yang sudah diinput

## Perubahan pada Kode
Perubahan yang dilakukan dari program awal:
Menambahkan variabel string untuk menyimpan nama.
Menggunakan getline(cin, nama) agar input bisa mengandung spasi.
Menambahkan pesan input: Silahkan inputkan nama Anda:.
Mengubah output menjadi dinamis: Hello <nama>.

## Hasil
Program berhasil menerima input dari pengguna dan menampilkan output sesuai dengan yang diharapkan. 
### Contoh
Input :
```
Ekal 
```
Output :
```
Hello Ekal
```
